# 轻泡自习（Bubble Study）

轻泡自习（Bubble Study）是一款极简、安静、带有仪式感的学习计时应用。

它不是复杂的任务管理工具，也不是充满排行榜和统计压力的番茄钟。打开应用后，屏幕中央只有一个梦幻海洋泡泡：轻触它，开始学习；再次轻触，结束学习。

## 核心体验

- 一个泡泡按钮，快速开始学习
- 泡泡和背景随学习时间逐渐变色
- 60 分钟到达最终颜色后不再继续变化
- 学习中 10 秒无操作后文字渐隐，保持纯净界面
- 从任意角落向对角线滑动，翻转泡泡查看累计学习时长
- 支持学习记录导出、导入与合并
- 支持 60 分钟振动提醒
- 学习中保持屏幕常亮
- 应用被系统清理后，重新打开可恢复正在学习状态
- 数据默认保存在本地，无账号压力

## 适合人群

- 想学习但难以启动的人
- 容易被手机分心的人
- 不喜欢复杂番茄钟和任务系统的人
- 希望用更柔和方式建立学习仪式感的人
- 想长期记录学习时间但不想被数据压迫的人

## 使用方法

1. 打开应用。
2. 轻触泡泡，开始学习。
3. 泡泡和背景会随着学习时间慢慢变色。
4. 学习中 10 秒无操作后，泡泡文字会渐隐。
5. 需要结束学习时，轻触泡泡让文字显示，再轻触一次结束。
6. 从屏幕任意角落向对角线滑动，可查看累计学习时长。

## 设置

长按泡泡 3 秒，或在 3 秒内连续轻触泡泡 5 次，可以进入设置页面。

设置中可以：

- 查看学习记录
- 导出学习数据
- 导入并合并旧数据
- 切换语言
- 开关 60 分钟振动提醒
- 恢复泡泡默认大小
- 清除所有数据

## 隐私

轻泡自习（Bubble Study）目前不需要账号，不主动上传学习记录。学习数据保存在设备本地；导出、导入文件均由用户主动操作。

详细说明见 [PRIVACY.md](PRIVACY.md)。
